Simple File Manager

Description

Simple File Manager is a Python project that performs basic file handling operations such as creating files, writing data into files, and reading file content.

Features

- Create a new file
- Write text into a file
- Read data from a file
- Display file content
- Menu-driven program

Technologies Used

- Python
- File Handling

Concepts Used

- Loops
- Conditional Statements
- Exception Handling
- File Operations

How to Run

1. Install Python
2. Run the program:

python file_manager.py
